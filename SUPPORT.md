# Support

## What This Repository Supports Today

This repository currently supports public discussion around:

1. Lime architecture and repository structure;
2. MicroPhoenix extraction boundaries for Lime;
3. GitHub publication readiness for the future standalone repository.

## What This Repository Does Not Support Yet

1. production runtime support;
2. operational incidents for a deployed service;
3. emergency response obligations;
4. private security reports in public issues;
5. sharing secrets or internal infrastructure data.

## Where To Start

Before opening a request, read:

1. [README.md](README.md)
2. [SECURITY.md](SECURITY.md)
3. [CONTRIBUTING.md](CONTRIBUTING.md)
4. [docs/LIME_STANDALONE_AI_CHAT_BACKEND_DESIGN.md](docs/LIME_STANDALONE_AI_CHAT_BACKEND_DESIGN.md)

## Response Expectations

This is currently a design and repository-seed project, not a staffed support desk.
Best-effort triage is the intended operating model until a maintainer team is declared.
